## 🎯 Project Summary

**Type:** Full-stack web application for EV diagnostic assistance

**Tech Stack:**
- Frontend: React 18 + Vite + Tailwind CSS
- Backend: Flask + Python
- Database: SQLite
- AI: BM25 + Groq API
- Deployment: Vercel + Render

**Key Innovation:** RAG Pipeline with optimistic message persistence

**Key Features:**
✅ User authentication with JWT
✅ RAG-powered diagnostic guidance
